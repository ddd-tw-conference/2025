# 技術棧

## 核心框架
- **Next.js 15.5.2** - React 框架 (export 模式)
- **React 19** - 前端框架
- **TypeScript** - 嚴格型別
- **Tailwind CSS** - 樣式框架
- **pnpm** - 包管理器

## 部署
- **GitHub Pages** - 靜態託管
- **SPA 模式** - 單頁應用
- **Static Export** - 靜態匯出

## 工具
- **Sharp** - 圖片優化
- **WebP** - 圖片格式
- **bundle analyzer** - 效能分析
