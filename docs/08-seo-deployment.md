## 🔍 SEO 優化策略

### 核心 SEO 元素

| 元素 | 重要性 | 實作方式 |
|------|--------|----------|
| **頁面標題** | 極高 | 每頁獨特的 `<title>` 標籤 |
| **描述標籤** | 高 | 吸引人的 `meta description` |
| **結構化資料** | 高 | JSON-LD 格式的事件資料 |
| **網址結構** | 中 | 語義化的 URL 路徑 |
| **內部連結** | 中 | 相關頁面的交叉連結 |

### 多語言 SEO

| 語言 | 網址結構 | 目標關鍵字 |
|------|----------|------------|
| **繁體中文** | `/` | DDD Taiwan 2025, 軟體設計, 領域驅動設計 |
| **英文** | `/en/` | DDD Taiwan 2025, Software Design, Domain-Driven Design |

### 結構化資料實作

使用 JSON-LD 格式提供活動資訊：

- **活動類型**：Conference
- **開始日期**：2025年活動日期
- **地點資訊**：完整地址和場地名稱
- **票價資訊**：各票種價格和可用性

## 📱 技術 SEO

### 頁面效能指標

| 指標 | 目標值 | 目前狀態 |
|------|--------|----------|
| **首次內容繪製 (FCP)** | < 1.8s | ✅ 已達標 |
| **最大內容繪製 (LCP)** | < 2.5s | ✅ 已達標 |
| **累積版面偏移 (CLS)** | < 0.1 | ✅ 已達標 |
| **首次輸入延遲 (FID)** | < 100ms | ✅ 已達標 |

### 圖片優化

- **格式選擇**：WebP 格式，回退至 PNG/JPEG
- **響應式圖片**：依螢幕大小載入適當尺寸
- **延遲載入**：非首屏圖片延遲載入
- **尺寸最佳化**：自動產生多種尺寸

### 字體優化

- **字體載入**：使用 `font-display: swap`
- **字體子集**：僅載入需要的字符
- **預載入**：關鍵字體提前載入

## ❓ 常見問答

**Q: 如何提升網站在 Google 的排名？**
A: 專注於內容品質、頁面效能、使用者體驗和技術 SEO。

**Q: 多語言網站如何避免重複內容問題？**
A: 使用 hreflang 標記和適當的 URL 結構。

**Q: 網站效能如何持續監控？**
A: 使用 Google PageSpeed Insights 和 Lighthouse 定期檢查。

**Q: 如何確保網站安全性？**
A: 使用 HTTPS、設定安全標頭、定期更新依賴套件。
